# Text

控制文本样式，这里主要是字体大小，字重，行高，间距，以及其它常用样式。

有关于文本颜色的内容请查看 Color 章节

TIP

不同于各 UI 库提供的文本排版工具，这些样式只需引用 class 类名，而不是载入一个组件

# 对照表

类名描述值依赖类名
fs-num设置字体大小 (8-60)numpx-
fw-num设置字重 (100-900)num-
lh-num设置行高 (8-60)numpx-
ls-num设置行高 (0-48)numpx-
text-left文本左对齐--
text-center文本居中对齐--
text-right文本右对齐--
text-ellipsis文本超出省略--
&.l-2文本超出省略 2 行-text-ellipsis
&.l-3文本超出省略 3 行-text-ellipsis
&.l-4文本超出省略 4 行-text-ellipsis
text-wrap文本换行--
text-nowrap文本强制不换行--
text-up字母全大写--
text-low字母全小写--
text-del文本删除线--
